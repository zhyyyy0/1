## 🎯 图片要求

- **格式**: JPG 或 PNG
- **尺寸**: 建议最小 800x600 像素
- **大小**: 单张不超过 5MB
- **质量**: 适合网页展示的清晰度

## 📝 注意事项

1. 请确保文件名与上面列出的完全一致
2. 某些图片文件名可能重复，这是正常的（同一张图片在不同分类中展示）
3. 如果某些图片暂时没有，网站仍能正常运行，只是对应位置会显示占位符

## 🚀 上传方法
